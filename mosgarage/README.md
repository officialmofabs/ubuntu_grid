MosGarage is a plug-and-play personal Microsoft Cloud dev garage with detachable modules.

## Quick Start

1. Clone/unzip the repo
2. Run:
```bash
chmod +x mosgarage.sh
./mosgarage.sh
```

3. Optional: start modules
```bash
./mosgarage.sh sharepoint powerapps teams functions
```

Access dashboard at: https://localhost
